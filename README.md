# Quad Tree

This project implements a Quad Tree data structure to assist in checking for collisons.

# Local Setup
```bash
git clone && cd $_
yarn
yarn build
http-server -p 3002 -c-1 // <- substitue your favorite local webserver if you do not have `http-server` installed
```
